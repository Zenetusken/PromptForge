# Auto Strategy

Analyze the prompt yourself and select the best optimization approach.

You have full discretion to combine techniques from any available strategy in the system. Examine the strategies directory for all installed strategies and their techniques.

Choose the approach that best addresses the prompt's weaknesses. You may combine multiple techniques from different strategies if appropriate. Focus on the highest-impact improvements first.

If the prompt is already strong, make minimal changes — do not over-optimize.
